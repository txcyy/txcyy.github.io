项目配置

tsconfig.json

问题一：'React' refers to a UMD global, but the current file is a module.

"compilerOptions": {
	...
    "jsx": "react-jsx"
  },

问题二：无法使用 JSX，除非提供了 "--jsx" 标志。

这是vscode typescript版本与项目上typescript版本不一致导致的，在vscode的右下角状态栏选择 Use workspace version解决。

开发

引入antd

antd默认按需引入

yarn add antd

引入@import '~antd/dist/antd.css';

支持less

yarn add less less-loader

在webpack.config.js中找到sass配置，参考sass配置less

问题一： TypeError: this.getOptions is not a function

原因是less-loader版本过高，卸载原来的版本，重新安装yarn add less-loader@6.0.0;

将chunk注入到html中

使用InlineChunkHtmlPlugin

new InlineChunkHtmlPlugin(HtmlWebpackPlugin, [/\.(js|css)$/])

使用外部资源减小体积

使用externals配置
externals: {
   xlsx: 'XLSX'
},
